Universal Commerce Protocol overview

How a chat request becomes a checkout

UCP gives AI agents and merchants a shared way to discover products, prepare carts, and hand buyers to a trusted checkout when payment or review is needed.

One-line takeaway: UCP lets the AI assist the shopping journey while the merchant keeps product truth, checkout policy, and payment trust.

Web search vs UCP search

Why Catalog changes how AI finds products

Without UCP, an agent mostly sees the web through search indexes. With Shopify Catalog, the agent can call a commerce API that returns product facts and the next buying action.

Key message: discovery and checkout become structured, not guessed
Without UCP

AI searches the crawled web

The agent meets pages that a search engine already indexed, then has to infer product facts from web content.

1
Search index first Results may reflect older crawled pages, not the current store state.
2
Store support is not enough A store-level UCP profile might only be seen if the agent opens that specific site.
3
Buying actions are unclear Web pages rarely expose a reliable machine-readable path for cart and checkout.
With Shopify Catalog + UCP

AI calls commerce directly

The agent queries Catalog instead of scraping search results, then receives product data and supported next steps.

1
Catalog API first Product, variant, image, price, availability, and shipping context can be returned directly.
2
More eligible to be selected Useful commerce facts that are hard to rank in web search become visible to the agent.
3
Next actions are explicit Catalog, cart, and checkout responses tell the agent what it can safely do next.
Fresher product truth Catalog can surface commerce data that search snippets and crawled pages may not keep current.
Better AI choice Availability, variants, and destination fit help the agent recommend products with less guesswork.
Actionable checkout path UCP can expose cart, checkout, and handoff steps instead of leaving the agent to scrape buttons.
One-line takeaway: Web search can discover pages; UCP lets the agent discover products and the next commerce action.
